# L4 vs L5 题型对比分析

## 📊 核心区别总结

| 维度 | L4 题型（标准车道） | L5 题型（闪电车道） |
|-----|-------------------|-------------------|
| **语境长度** | 短语（Phrase） | 完整句子（Sentence） |
| **难度等级** | ⭐⭐⭐ 中等 | ⭐⭐⭐⭐⭐ 高难度 |
| **测试目标** | 语块拼写（Phrase Spelling） | 能力迁移（Transfer） |
| **适用人群** | 核心词（常态） | 核心词（优等生） |
| **复习曲线** | 1→3→7→15 天 | 1→15 天（减负50%） |
| **理论支撑** | 间隔重复（Spaced Repetition） | 必要的困难（Desirable Difficulties） |

## 🎯 实际题目示例

### 示例单词：`adapt` (适应)

#### L4 题型 - 短语拼写
```
📘 标准车道 | MODE: L4

显示的语境：
____ to new changes

中文释义：
v. (使)适应

要求拼写：adapt
```

**特点：**
- 使用短语：`adapt to new changes`（适应新的变化）
- 语境较短，3-5个单词
- 固定搭配，降低认知负荷
- 聚焦于"语块记忆"

---

#### L5 题型 - 句子迁移
```
⚡ 闪电车道 | MODE: L5

显示的语境：
It takes time to ____ to a new school.

中文释义：
v. (使)适应

要求拼写：adapt
```

**特点：**
- 使用完整句子：`It takes time to adapt to a new school.`（适应新学校需要时间）
- 语境较长，7-10个单词
- 需要理解整句意思
- 测试真实语境中的应用能力（迁移）

## 💡 设计理念对比

### L4 - 语块拼写（Chunk-Based）
```
理念：通过高频搭配强化记忆

adapt to new changes
    ↓
记忆单元：adapt to + 名词
固定搭配：便于提取

认知负荷：★★★☆☆（中等）
记忆强度：★★★★☆（较强）
```

### L5 - 能力迁移（Transfer-Based）
```
理念：验证深层理解和灵活应用

It takes time to adapt to a new school.
    ↓
需要理解：整句逻辑 + 语法结构 + 单词用法
灵活应用：不同语境中使用

认知负荷：★★★★★（高）
记忆强度：★★★★★（最强）
```

## 🔍 代码实现对比

### 当前代码逻辑
```javascript
// SmartReview.jsx 第 196-199 行

// L4模式使用短语，L5模式使用长句子
const sentence = currentLevel === 'L4' 
  ? (currentWord.context?.[0]?.phrase || currentWord.phrase || currentWord.context?.[0]?.sentence || '')
  : (currentWord.context?.[0]?.sentence || currentWord.phrase || '');
```

**解析：**
- **L4 优先级**：`phrase` → `context[0].phrase` → `context[0].sentence`（降级）
- **L5 优先级**：`sentence` → `context[0].sentence` → `phrase`（降级）

## 🧠 认知科学分析

### L4 的优势
1. **降低认知负荷**
   - 短语长度适中（3-5词）
   - 固定搭配易于记忆
   - 适合初次掌握和巩固

2. **符合语块理论（Chunking）**
   - 人脑记忆以"组块"为单位
   - `adapt to` 作为固定搭配更易提取
   - 提高流利度

3. **适用场景**
   - Phase 3 验收后首次复习
   - 答错后降级复习
   - 确保基础牢固

### L5 的优势
1. **提高必要困难（Desirable Difficulties）**
   - 更长的语境增加提取难度
   - 需要整合多重信息
   - 产生更深记忆痕迹

2. **验证迁移能力**
   - 不同于学习时的语境
   - 测试真实应用场景
   - 确认深层掌握

3. **适用场景**
   - Day 1 能力测试
   - 优等生快速通道
   - 减少机械重复

## 🎯 选择建议

### 适合选择标准车道（L4）的情况
- ✅ 首次复习核心词
- ✅ 需要稳扎稳打建立信心
- ✅ 对单词掌握程度不确定
- ✅ 希望通过固定搭配强化记忆
- ✅ 时间充裕，可以多次复习

### 适合选择闪电车道（L5）的情况
- ✅ 对单词已有较好掌握
- ✅ 希望挑战更高难度
- ✅ 想要快速通过复习（减负50%）
- ✅ 需要验证迁移能力
- ✅ 追求高效学习

## 📊 效果评估

### L4 预期效果
- 通过率：~70-80%
- 平均用时：15-20秒/题
- 记忆保持：良好（需多次复习）
- 学习感受：稳定、可控

### L5 预期效果
- 通过率：~50-60%
- 平均用时：20-30秒/题
- 记忆保持：优秀（深度加工）
- 学习感受：挑战、成就感高

## 🔧 技术实现细节

### 完整数据流
```javascript
1. 用户选择车道
   → ReviewSelection: handleTrackSelection('standard' | 'fast')

2. 传递参数
   → navigate('/smart-review', { 
       state: { 
         track: 'standard',  // 或 'fast'
         level: 'L4'         // 或 'L5'
       }
     })

3. SmartReview 接收
   → const { track, level } = location.state || {}

4. 动态题型
   → const currentLevel = getCurrentLevel()
   → L4: phrase / L5: sentence

5. 显示内容
   → const sentence = currentLevel === 'L4' 
        ? phrase 
        : sentence
   → displaySentence = sentence.replace(word, '____')
```

## 总结

### 核心差异
1. **语境长度**：L4 短语 vs L5 句子
2. **难度定位**：L4 巩固 vs L5 挑战
3. **测试目标**：L4 语块 vs L5 迁移
4. **适用人群**：L4 常态 vs L5 优等生

### 协同作用
- L4 和 L5 不是对立而是互补
- L5 失败后降级到 L4 保证掌握
- 两者共同构成完整的复习体系
- 满足不同水平学生的需求
