## 三轨制矩阵对照表

| 轨道类型 | 适用对象 | 触发条件 | 复习曲线 | 题型 | 减负比例 |
|---------|---------|---------|---------|------|---------|
| **轻量车道** (L2形-义) | 非核心词<br/>Tag: Non-Core | Phase 3 通过 L4 | Day 1→3→15<br/>（稀疏打卡） | L1 语境探针<br/>（语块选义） | 50% |
| **标准车道** (核心L4) | ⭐ 核心词<br/>Tag: Core（常态） | Day 1 L5测试<br/>失败或求助 | 1→3→7→15<br/>（强艾宾浩斯） | L4 全拼写<br/>（语块盲打） | - |
| **闪电车道** (核心L5) | ⭐ 核心词<br/>Tag: Core（优等生） | Day 1 L5迁移<br/>一次通过 | Day 1→15<br/>（跳过Day 3、7） | L5 全拼写<br/>（短句迁移） | 50% |

## 理论支撑对照

```
┌────────────────────────────────────────────────────────────────┐
│                         理论依据                                │
├────────────┬───────────────────────────────────────────────────┤
│ 轻量车道    │ 启动效应 (Priming Effect)                         │
│            │ • 对消极词汇，只需低强度语境刺激维持再认能力         │
│            │ • 不值得占用过多心智资源                           │
├────────────┼───────────────────────────────────────────────────┤
│ 标准车道    │ 间隔重复 (Spaced Repetition)                      │
│            │ • 对核心生产性词汇，必须严格遵守遗忘曲线             │
│            │ • 1→3→7→15 确保高强度加固                         │
├────────────┼───────────────────────────────────────────────────┤
│ 闪电车道    │ 必要的困难 (Desirable Difficulties)               │
│            │ • Bjork理论：提高初始提取难度产生更强记忆保持        │
│            │ • 直接考迁移验证深层掌握度，剔除无效重复             │
└────────────┴───────────────────────────────────────────────────┘
```

## 数据流向

```
User Click (选择车道)
    ↓
ReviewSelection: handleTrackSelection(track)
    ↓
navigate('/smart-review', { 
  state: { 
    track: 'standard' | 'fast',
    level: 'L4' | 'L5'
  }
})
    ↓
SmartReview receives location.state:
    {
      track: 'standard' | 'fast',
      level: 'L4' | 'L5'
    }
    ↓
SmartReview 逻辑处理:
    • getCurrentLevel():
      - appearanceCount > 0 → 返回 'L4'（降级）
      - appearanceCount === 0 → 返回 level（首次）
    
    • 题目内容选择:
      - L4: 使用 phrase（短语）
      - L5: 使用 sentence（完整句子）
    
    • Header 显示:
      - track → 车道徽章（📘 标准车道 / ⚡ 闪电车道）
      - currentLevel → 模式徽章（MODE: L4 / MODE: L5）
      - progress → 进度徽章（1 / 3）
```
