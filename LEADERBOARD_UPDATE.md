## 🆚 两栏设计

### 左栏：班级排行榜（400px）

#### 1. 我的排名卡片（高亮）
```
┌────────────────────────┐
│ 第3名  我              │
│        5 已掌握         │
│        3 复习中    56% │
└────────────────────────┘
```
- 白色背景突出显示
- 金色边框高亮
- 大号百分比显示掌握度
- 绿色和黄色标签显示详细状态

#### 2. 排行榜列表
```
🥇 张小明  9/9 掌握   100%
🥈 李华    7/9 掌握    78%
🥉 我      5/9 掌握    56%  ← 高亮
4  王芳    4/9 掌握    44%
5  赵强    3/9 掌握    33%
```
- 前3名使用奖牌图标
- 当前学生行高亮
- 玻璃态半透明卡片
- 悬停时向右滑动

### 右栏：最近学习的单词

#### 单词卡片网格（2列）
```
┌──────────┐  ┌──────────┐
│ adapt  ✓ │  │ word   ○ │
│ (使)适应 │  │  释义    │
└──────────┘  └──────────┘
```

**特点：**
- 2x3 网格，显示最近6个单词
- 颜色编码：
  - 绿色边框 + ✓ = 已掌握
  - 黄色边框 + ○ = 复习中
  - 红色边框 + ! = 需加强
- 悬停上浮效果
- 点击可查看详情（未来功能）

## 💡 设计理念

### 1. 社会竞争（Social Competition）

**元素：**
- 班级排行榜
- 实时排名显示
- 百分比对比

**心理效应：**
- 激发竞争意识
- "我要超过前面的同学"
- 社会压力转化为学习动力

**注意事项：**
- 避免过度竞争压力
- 强调个人进步而非绝对排名
- 排行榜数据可选择是否公开

### 2. 即时反馈（Instant Feedback）

**"我的排名卡片"设计：**
```
第3名 ← 明确的位置
我    ← 个人化标识
56%   ← 量化的成绩
```

**效果：**
- 一眼看到自己的位置
- 清楚了解与他人的差距
- 激励继续努力

### 3. 学习轨迹（Learning Track）

**"最近学习"区域：**
- 展示最近学习的6个单词
- 状态一目了然（绿/黄/红）
- 复习进度可视化

**价值：**
- 回顾最近学习内容
- 快速识别需要加强的单词
- 建立学习成就感

## 📊 数据展示

### 排行榜数据结构
```javascript
{
  rank: 3,           // 排名
  name: "我",        // 学生名称
  mastered: 5,       // 已掌握数量
  total: 9,          // 总单词数
  percentage: 56     // 掌握率
}
```

### 动态计算
- 掌握率：`(已掌握 / 总数) × 100%`
- 排名：基于掌握率降序排列
- 实时更新：学习后自动刷新

## ⚠️ 注意事项

### 1. 数据隐私
- 排行榜数据应可选择匿名显示
- 家长/老师可设置是否公开排名
- 避免过度竞争压力

### 2. 正向激励
- 强调个人进步而非绝对排名
- 提供"进步最快"等多维度排名
- 避免负面标签（如"最差"）

### 3. 公平性
- 考虑不同班级/年级的差异
- 可按难度分级排名
- 提供同等级对比
