## 🔄 答题流程对比

### L4 流程
```
1. 看到短语提示
   "adapt to new changes"
        ↓
2. 识别固定搭配
   "adapt to" (熟悉的组合)
        ↓
3. 回忆拼写
   a-d-a-p-t (5个字母)
        ↓
4. 输入验证
   ✓ 正确 → 完成
   ✗ 错误 → 重试/降级
```

### L5 流程
```
1. 阅读完整句子
   "It takes time to ____ to a new school."
        ↓
2. 理解句意
   "适应新学校需要时间"
        ↓
3. 分析语法
   "It takes time to do sth."
   "to ____ to a new school"
        ↓
4. 确定单词
   "adapt" (适应)
        ↓
5. 回忆拼写
   a-d-a-p-t (5个字母)
        ↓
6. 输入验证
   ✓ 正确 → 完成（高成就感）
   ✗ 错误 → 降级为 L4
```

## 总结

### 关键差异点
1. **长度**：L4 短语（3词） vs L5 句子（9词）
2. **难度**：L4 中等 vs L5 高
3. **目标**：L4 语块 vs L5 迁移
4. **体验**：L4 稳定 vs L5 挑战

### 协同效果
```
L5 挑战
   ↓ 失败
L4 巩固
   ↓ 通过
✓ 掌握
```

两者不是替代而是互补，共同构成完整的学习梯度！
